# AI 学习网站 - 功能列表

在线地址：https://ai-learning-site-silk.vercel.app

## 核心功能（全部已实现）

### 课程系统
- [x] 3 门默认课程（编程基础 8 章、AI 知识 6 章、AI Agent 开发 6 章）
- [x] AI 自动生成教学文档（Markdown 格式，DeepSeek API）
- [x] 文档渲染（react-markdown + remark-gfm）
- [x] 学习进度追踪（章节完成标记 + 进度百分比）
- [x] 首页仪表盘：总进度条、继续学习、最近学习、AI 建议
- [x] 侧边栏导航（展开/折叠，桌面+移动端响应式）

### 课程管理 /admin/courses
- [x] 添加/编辑/删除课程（标题、图标、描述、领域、难度）

- [x] 修改后侧边栏/首页/课程页实时同步

## API 路由

| 路由 | 方法 | 用途 |
|------|------|------|
| /api/chat | POST | AI 对话（支持联网搜索） |
| /api/generate-doc | POST | 生成教学文档 |
| /api/quiz | POST | 生成测验题目 |
| /api/mindmap | POST | 生成思维导图（含对话内容） |
| /api/summary | POST | 生成月度总结 |
| /api/knowledge-cards | POST | 提取知识卡片 |
| /api/search | POST | DuckDuckGo 搜索 |
| /api/data | POST | 云端数据 CRUD（Vercel KV） |

## 数据存储

所有数据双写：localStorage（本地缓存）+ Vercel KV（云端持久化）

- 课程数据、学习进度、对话会话
- 用户设置、知识卡片、提示词配置
- 月度总结、思维导图缓存、文档缓存
